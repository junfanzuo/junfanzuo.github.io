Logic II

28 Dec 2025

Logic I covered the truth table. We can tell an argument is valid by constructing a truth table where no row has true premises but a false conclusion. This post introduces Natural Deduction, another method to determine the validity of an argument. We can prove that (a valid) argument is valid. My memory of Natural Deduction is a bit rusty so the best way for you the reader to fully grasp Natural Deduction is to do practice exercises – you can find them in the textbook. This post simply summarises Natural Deduction rules.

There are three kinds of rules. Basic Rules are all you need: if an argument is valid, you can simply use Basic Rules to prove it. Derived Rules are not necessary but they will be convenient for proof in some cases. Replacement Rules allow you to replace a sentence part, instead of the whole sentence. The following are the demonstrations of these rules which are adapted from Ichikawa and Magnus (2019, pp. 110-123)

§1.3 Replacement Rules

Commutativity
(Φ ∧ Ψ) ↔ (Ψ ∧ Φ)
(Φ ∨ Ψ) ↔ (Ψ ∨ Φ)
(Φ ↔ Ψ) ↔ (Ψ ↔ Φ) Comm
Double Negation
~~Φ ↔ Φ DN
De Morgan's Laws
~(Φ ∨ Ψ) ↔ (~Φ ∧ ~Ψ)
~(Φ ∧ Ψ) ↔ (~Φ ∨ ~Ψ) DeM
Material Conditional
(Φ → Ψ) ↔ (~Φ ∨ Ψ)
(Φ ∨ Ψ) ↔ (~Φ → Ψ) MC
Biconditional Exchange
[(Φ → Ψ) ∧ (Ψ → Φ)] ↔ (Φ ↔ Ψ) ↔ex
